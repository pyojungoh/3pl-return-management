# 로고 업로드 가이드

## 로고 파일 요구사항

### 1. 파일 형식
- **권장**: PNG (투명 배경)
- **대안**: SVG (벡터 형식, 확대 시 깨지지 않음)
- **비권장**: JPG (투명 배경 불가)

### 2. 파일 크기
- **권장 크기**: 200x200px ~ 400x400px (정사각형)
- **최소 크기**: 120x120px
- **최대 크기**: 800x800px
- **비율**: 1:1 (정사각형) 권장

### 3. 배경
- **투명 배경 필수** (PNG with alpha channel)
- 로고만 표시되고 배경은 투명해야 함

### 4. 파일명
- `logo.png` 또는 `logo.svg`
- 소문자로 작성

## 업로드 방법

### 1. 파일 준비
로고 파일을 다음 사양으로 준비하세요:
- 파일명: `logo.png` 또는 `logo.svg`
- 크기: 200x200px ~ 400x400px
- 형식: PNG (투명 배경) 또는 SVG
- 위치: 프로젝트의 `static` 폴더

### 2. 업로드 위치
```
프로젝트 루트/
  └── static/
      └── logo.png  (또는 logo.svg)
```

### 3. 파일 경로
업로드 후 자동으로 다음 경로에서 접근 가능합니다:
- URL: `https://jjaysolution.com/static/logo.png`
- 또는: `https://jjaysolution.com/static/logo.svg`

## 로고 디자인 팁

### 권장 사항
1. **단순한 디자인**: 작은 크기에서도 잘 보이도록
2. **높은 대비**: 오렌지 그라데이션 배경에서 잘 보이도록
3. **중앙 정렬**: 원형 프레임 중앙에 배치
4. **여백**: 로고 주변에 충분한 여백 (padding 15px)

### 색상 권장사항
- 흰색 또는 밝은 색상 (오렌지 배경과 대비)
- 또는 오렌지 계열 색상 (그라데이션과 조화)

## 로고가 없을 경우

로고 파일이 없거나 로드에 실패하면 자동으로 "JJ" 텍스트가 표시됩니다.

## 확인 방법

1. 로고 파일을 `static/logo.png`에 업로드
2. 브라우저에서 `https://jjaysolution.com/static/logo.png` 접속하여 확인
3. 로그인 페이지에서 로고가 표시되는지 확인

## 예시

### 좋은 예
- ✅ 300x300px PNG, 투명 배경
- ✅ 200x200px SVG
- ✅ 흰색 또는 밝은 색상 로고

### 나쁜 예
- ❌ JPG 형식 (투명 배경 불가)
- ❌ 50x50px (너무 작음)
- ❌ 2000x2000px (너무 큼, 로딩 느림)
- ❌ 검은색 배경 (오렌지 배경과 대비 부족)
