========================================
데이터 마이그레이션 간단 방법
========================================

1. Vercel 대시보드에서 DATABASE_URL 복사
   - Vercel → 프로젝트 선택
   - Settings → Environment Variables
   - DATABASE_URL 값 복사

2. CMD 창에서 실행

   set DATABASE_URL=여기에_복사한_URL_붙여넣기
   python migrate_sqlite_to_postgresql.py

3. 마이그레이션 완료 확인
   - 화주사 계정 개수 확인
   - 반품 데이터 개수 확인

4. 웹사이트에서 확인
   - Vercel 배포 URL 접속
   - 관리자 계정으로 로그인 (admin/admin123)
   - 데이터 확인

========================================
