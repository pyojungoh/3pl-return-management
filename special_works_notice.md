# 🔧 특수작업 안내

## 📋 특수작업이란?

특수작업은 일반적인 물류 작업과는 별도로 추가적인 작업이 필요한 경우에 발생하는 작업입니다.

예를 들어:
- 특별한 포장 작업
- 제품 재검수
- 라벨 재부착
- 특수 배송 준비
- 기타 추가 작업

이러한 특수작업은 **별도의 비용**이 발생하며, 작업 내역과 비용을 확인하실 수 있습니다.

---

## 🎯 화주사 모드 사용 방법

### 1. 특수작업 메뉴 접근
- 메인 메뉴에서 **"🔧 특수작업"** 버튼을 클릭하세요.

### 2. 월별 조회
- 화면 상단에 **"조회 월"** 선택 메뉴가 있습니다.
- 원하는 연도와 월을 선택한 후 **"조회"** 버튼을 클릭하세요.
- 기본적으로 현재 월이 선택되어 있습니다.

### 3. 작업 종류별 통계 확인
- 조회 결과 상단에 **작업 종류별 통계 박스**가 표시됩니다.
- 각 박스에는 다음 정보가 표시됩니다:
  - **작업 종류명** (예: 포장작업, 검수작업 등)
  - **작업 건수**
  - **총액** (해당 작업 종류의 총 비용)
- 특정 작업 종류 박스를 클릭하면 해당 작업 종류만 필터링하여 볼 수 있습니다.
- 같은 박스를 다시 클릭하면 필터가 해제됩니다.

### 4. 작업 내역 확인
조회 결과 테이블에서 다음 정보를 확인하실 수 있습니다:

| 항목 | 설명 |
|------|------|
| 순번 | 작업 순서 번호 |
| 작업 일자 | 작업이 수행된 날짜 |
| 작업 종류 | 수행된 작업의 종류 |
| 수량 | 작업 수량 |
| 단가 | 작업 1건당 단가 (원) |
| 총액 | 수량 × 단가 = 총 비용 (원) |
| 사진 | 작업 관련 사진 (있는 경우) |
| 메모 | 작업 관련 메모 및 특이사항 |

### 5. 작업 종류별 필터링
- 통계 박스를 클릭하면 해당 작업 종류의 작업만 표시됩니다.
- 필터를 해제하려면 같은 박스를 다시 클릭하세요.

---

## 💡 유용한 팁

1. **월별로 조회**: 각 월별로 특수작업 내역을 조회하여 비용을 확인할 수 있습니다.

2. **작업 종류별 확인**: 통계 박스를 활용하여 어떤 작업이 얼마나 발생했는지 한눈에 확인하세요.

3. **사진 확인**: 작업 관련 사진이 있는 경우 확인하여 작업 내용을 더 명확히 이해할 수 있습니다.

4. **메모 확인**: 작업 관련 특이사항이나 메모가 있으면 확인하세요.

---

## ⚠️ 주의사항

- 특수작업은 **일반 물류 작업과 별도로 비용이 발생**합니다.
- 작업 내역과 비용은 관리자가 등록한 내용을 기준으로 표시됩니다.
- 비용 관련 문의사항이 있으시면 관리자에게 연락해 주세요.
