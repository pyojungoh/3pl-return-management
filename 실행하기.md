# ⚡ 빠른 실행 가이드

## 방법 1: 배치 파일 사용 (가장 쉬움)

1. `로컬실행.bat` 파일을 더블클릭
2. 자동으로 서버가 실행됩니다
3. 브라우저에서 http://localhost:5000 접속

## 방법 2: CMD에서 직접 실행

1. **CMD 열기**
   - Windows 키 + R
   - `cmd` 입력 후 Enter

2. **프로젝트 폴더로 이동**
```cmd
cd C:\3pl반품관리및화주사관리
```

3. **가상환경 활성화 (있는 경우)**
```cmd
venv\Scripts\activate
```

4. **패키지 설치 (처음만)**
```cmd
pip install -r requirements.txt
```

5. **서버 실행**
```cmd
python app.py
```

6. **브라우저에서 접속**
   - http://localhost:5000

## 방법 3: PowerShell에서 실행

1. **프로젝트 폴더에서 Shift + 우클릭**
2. **"PowerShell 창에서 열기" 선택**
3. **실행:**
```powershell
python app.py
```

## 문제 해결

### Python이 없다는 오류
- Python 설치: https://www.python.org/downloads/
- 설치 시 "Add Python to PATH" 체크 필수!

### 포트 사용 중 오류
- `app.py` 파일을 열어서 포트 번호 변경 (115번째 줄 근처)
- 기본: `port = 5000` → 다른 번호로 변경 (예: `5001`)

### 패키지 설치 오류
```cmd
pip install --upgrade pip
pip install -r requirements.txt
```

### 가상환경 문제
- 가상환경 없이 실행해도 됩니다
- 직접 `pip install -r requirements.txt` 실행

## 서버 중지
- CMD/PowerShell에서 `Ctrl + C` 누르기
