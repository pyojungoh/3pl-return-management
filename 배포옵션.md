# 🚀 배포 옵션 가이드

## 1. 로컬 실행 (가장 간단 - 테스트용)

### Windows에서 실행
1. `로컬실행.bat` 파일을 더블클릭
2. 또는 CMD/PowerShell에서:
```cmd
python app.py
```
3. 브라우저에서 http://localhost:5000 접속

### 필요사항
- Python 3.7 이상 설치
- `pip install -r requirements.txt` 실행

---

## 2. Vercel 배포 (무료, 이미 설정 파일 있음) ⭐ 추천

### 장점
- 무료 플랜 제공
- GitHub 연동으로 자동 배포
- 이미 `vercel.json` 설정 파일이 있음
- 빠른 배포 속도

### 배포 방법
1. **GitHub에 코드 업로드**
   - GitHub 저장소 생성
   - 파일 업로드 (venv/, data.db 제외)

2. **Vercel 연결**
   - https://vercel.com 접속
   - GitHub 계정으로 로그인
   - "New Project" 클릭
   - 저장소 선택
   - "Deploy" 클릭

3. **완료**
   - 자동으로 배포됨
   - URL 생성 (예: `your-app.vercel.app`)

### 주의사항
- SQLite 데이터베이스는 `/tmp` 디렉토리 사용 (임시)
- 서버리스 환경이므로 장기 실행 불가

---

## 3. Render 배포 (무료, 간단)

### 장점
- 무료 플랜 제공
- GitHub 연동
- 영구 웹 서비스 지원
- SQLite 데이터베이스 지원

### 배포 방법
1. **GitHub에 코드 업로드**

2. **Render 연결**
   - https://render.com 접속
   - GitHub 계정으로 로그인
   - "New +" → "Web Service" 클릭
   - 저장소 선택

3. **설정**
   - Name: 프로젝트 이름
   - Environment: Python 3
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app --bind 0.0.0.0:$PORT`

4. **완료**
   - 배포 완료 후 URL 생성

### 필요 파일
- `requirements.txt` ✅ (있음)
- `Procfile` 또는 Start Command 설정

---

## 4. PythonAnywhere 배포 (무료, Python 전용)

### 장점
- Python 전용 호스팅
- 무료 플랜 제공
- 쉬운 설정

### 배포 방법
1. https://www.pythonanywhere.com 회원가입
2. "Files" 탭에서 코드 업로드
3. "Web" 탭에서 Flask 앱 생성
4. WSGI 파일 설정

---

## 5. Google Apps Script (현재 사용 중인 것 같음)

### 확인사항
- `Code.js` 파일이 있음
- `appsscript.json` 설정 파일이 있음
- Google Apps Script에서 이미 배포되어 있을 수 있음

### 사용 방법
1. https://script.google.com 접속
2. 프로젝트 열기
3. "배포" → "웹 앱으로 배포"
4. URL 확인

## 💡 PowerShell/CMD 실행 문제 해결

### 방법 1: CMD 사용
1. Windows 키 + R
2. `cmd` 입력 후 Enter
3. 프로젝트 폴더로 이동:
```cmd
cd C:\3pl반품관리및화주사관리
```
4. 실행:
```cmd
python app.py
```

### 방법 2: 배치 파일 사용
1. `로컬실행.bat` 파일 더블클릭
2. 자동으로 실행됨

### 방법 3: 직접 Python 실행
1. 프로젝트 폴더에서 Shift + 우클릭
2. "PowerShell 창에서 열기" 또는 "터미널에서 열기"
3. 실행:
```powershell
python app.py
```

---

## 🔧 문제 해결

### Python이 설치되지 않은 경우
1. https://www.python.org/downloads/ 에서 Python 다운로드
2. 설치 시 "Add Python to PATH" 체크

### 패키지 설치 오류
```cmd
pip install --upgrade pip
pip install -r requirements.txt
```

### 포트 사용 중 오류
- `app.py`에서 포트 번호 변경 (기본: 5000)
