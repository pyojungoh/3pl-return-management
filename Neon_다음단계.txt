[다음 단계]

1단계: DATABASE_URL 복사
1. Quickstart 섹션에서
2. "Show secret" 클릭 (비밀번호 보기)
3. "Copy Snippet" 클릭 (전체 코드 복사)
   또는 DATABASE_URL 값만 복사

2단계: Vercel 환경 변수 설정
1. Vercel 대시보드 → 프로젝트 → Settings → Environment Variables
2. "New Variable" 클릭
3. 입력:
   - Name: DATABASE_URL
   - Value: 복사한 DATABASE_URL 값
   - Environment: Production, Preview, Development 모두 선택
4. "Save" 클릭

3단계: 코드 전환
→ 제가 PostgreSQL로 전환해드리겠습니다.
→ 준비되면 "코드 전환해줘"라고 말씀해주세요.

==========================================
[중요]
==========================================

DATABASE_URL은 이런 형식입니다:
postgresql://user:password@host:5432/database?sslmode=require

이 값을 Vercel 환경 변수에 설정하면 됩니다!
