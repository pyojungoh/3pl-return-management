# Vercel 데이터베이스 선택 가이드

## 추천: Neon 선택 ⭐

**Neon - Serverless Postgres**를 선택하세요!

### 이유:
- ✅ Serverless Postgres (서버리스 최적화)
- ✅ 무료 플랜 제공
- ✅ Vercel과 완벽 통합
- ✅ 자동 환경 변수 설정
- ✅ 빠른 설정

## 선택 방법

1. **Marketplace Database Providers** 섹션에서
2. **Neon** 클릭
3. **"Continue"** 클릭

## 대안: Supabase

Neon이 안 되면 **Supabase**도 좋은 선택입니다:
- ✅ Postgres backend
- ✅ 무료 플랜 제공
- ✅ 쉬운 설정

## 다음 단계

Neon 선택 후:
1. Neon 계정 연결 (GitHub 로그인)
2. 프로젝트 이름 입력
3. 리전 선택 (가장 가까운 곳)
4. 데이터베이스 생성
5. 환경 변수 자동 설정됨

## 완료!

환경 변수 `DATABASE_URL` 또는 `POSTGRES_URL`이 자동으로 설정됩니다!
