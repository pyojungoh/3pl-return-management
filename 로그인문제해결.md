# 로그인 문제 해결 방법

## 문제 원인
Vercel에 배포한 후 데이터베이스가 비어있어서 사용자 계정이 없습니다.

## 해결 방법 1: 웹에서 관리자 계정 생성 (가장 간단)

### 방법 A: 임시 API 엔드포인트 사용
배포된 사이트에서 브라우저 개발자 도구(F12) 콘솔에서 실행:

```javascript
fetch('/api/auth/register', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    company_name: '관리자',
    username: 'admin',
    password: 'admin123',
    role: '관리자'
  })
})
.then(r => r.json())
.then(data => console.log(data));
```

### 방법 B: 로컬에서 실행 후 data.db 업로드
1. 로컬에서 `create_admin.py` 실행:
```cmd
python create_admin.py
```

2. 생성된 `data.db` 파일을 GitHub에 커밋하고 푸시
3. Vercel에서 자동 재배포

## 해결 방법 2: Vercel 환경 변수로 초기 계정 생성

`app.py`에 초기화 스크립트 추가 (권장하지 않음 - 보안 문제)

## 해결 방법 3: 관리자 등록 API 직접 호출

Postman이나 curl로 직접 호출:

```bash
curl -X POST https://your-app.vercel.app/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "company_name": "관리자",
    "username": "admin",
    "password": "admin123",
    "role": "관리자"
  }'
```

## 추천 방법
**방법 1-A (브라우저 콘솔)**가 가장 빠르고 간단합니다!
