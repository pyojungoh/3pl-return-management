==========================================
GitHub 푸시 완료하기
==========================================

[현재 상황]
- .gitignore가 수정되었지만 아직 커밋 안 됨
- data.db가 추가되지 않음

[해결 방법 - 순서대로 실행]

1단계: .gitignore 먼저 커밋
git add .gitignore
git commit -m ".gitignore 수정 - data.db 포함"

2단계: data.db 강제 추가
git add -f data.db

3단계: data.db 커밋
git commit -m "데이터베이스 마이그레이션 완료"

4단계: 푸시
git push origin main

==========================================
[한 번에 실행]
==========================================

git add .gitignore
git commit -m ".gitignore 수정"
git add -f data.db
git commit -m "데이터베이스 마이그레이션 완료"
git push origin main

==========================================
[확인]
==========================================

푸시 후 GitHub에서 확인:
1. GitHub 저장소 접속
2. data.db 파일이 있는지 확인
3. 파일 크기가 0이 아니면 성공
