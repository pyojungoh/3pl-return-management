==========================================
Vercel 환경 변수 추가 방법
==========================================

[1단계] Key 입력
1. "Key" 필드 클릭
2. "CLIENT_KEY..." 텍스트 삭제
3. "DATABASE_URL" 입력

[2단계] Value 입력
1. "Value" 필드 클릭
2. Neon에서 복사한 DATABASE_URL 값 붙여넣기
   (postgresql://로 시작하는 전체 문자열)

[3단계] Environments 확인
- "Production, Preview, and Development"가 선택되어 있는지 확인
- 그대로 두면 됨 (모든 환경에 적용)

[4단계] Link To Projects (프로젝트 연결)
1. "Link To Projects" 필드 클릭
2. 프로젝트 검색 또는 선택
3. "3pl-return-management" 프로젝트 선택

[5단계] Save 클릭
- 모든 입력 완료 후
- "Save" 버튼 클릭

==========================================
[주의사항]
==========================================

- DATABASE_URL 값은 postgresql://로 시작하는 전체 문자열
- Neon에서 "Copy Snippet"으로 복사한 값 사용
- 프로젝트에 반드시 연결해야 함
