==========================================
로컬 서버 실행 명령어 (복사해서 사용)
==========================================

[1] 프로젝트 폴더로 이동
cd C:\3pl반품관리및화주사관리

[2] 가상환경 활성화 (있는 경우)
venv\Scripts\activate

[3] 패키지 설치 (처음만 또는 업데이트 시)
pip install -r requirements.txt

[4] 서버 실행
python app.py

==========================================
서버 실행 후 브라우저에서 접속:
http://localhost:5000

서버 중지: Ctrl + C
==========================================
